# ⚡ Estudos de Bootstrap 5

Este repositório reúne exercícios e projetos desenvolvidos durante meus estudos de Bootstrap 5, com foco em responsividade, componentes e construção de interfaces modernas.

## 📝 O que foi praticado

* **Sistema de Grids**: Utilização de `container`, `row` e `col` para criar layouts responsivos.
* **Componentes**: Desenvolvimento de navbars, botões, formulários, cards e carrosséis.
* **Tipografia e Cores**: Uso de classes utilitárias para estilização de textos e elementos.
* **Responsividade**: Aplicação de breakpoints (`sm`, `md`, `lg`, `xl`, `xxl`) para adaptação em diferentes dispositivos.
* **Flexbox**: Utilização de utilitários do Bootstrap para alinhamento e distribuição de elementos.

## 🚀 Projeto em Destaque: Finans

O principal projeto desenvolvido nesta etapa foi o **Projeto Finans**, uma landing page para uma empresa de serviços financeiros.

### Destaques do Projeto

* Navegação fixa
* Layout responsivo
* Grid adaptativo
* Componentes reutilizáveis
* Estilização personalizada com CSS

## 🛠️ Como visualizar os projetos

1. Clone o repositório:

```bash
git clone git@github.com:manoela-alves-silva/bootstrap-estudos.git
```

2. Abra os arquivos `.html` no navegador.
